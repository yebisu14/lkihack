VRChat向け　三色串団子 紹介文
VRChat向けに作成された三色串団子です。
VRChat外での利用も可能です。
全ての団子に別のマテリアルが割り当てられているので、unity上で簡単にそれぞれの色を変更できます。

内容物
３Dモデル（FBX）
テクスチャ
readme

ポリゴン数
3066

導入方法
アバターをすでに読み込んだunityのプロジェクトにunitypackageを読み込んだあと、本モデルをシーンに追加し、手のボーンに直接組み込んでください。
串の先と一つ目の団子にシェイプキーが仕込んであるのでご自由にお使いください。
